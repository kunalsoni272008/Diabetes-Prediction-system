# Diabetes Prediction Using Machine Learning

## Project Overview
This project predicts whether a person is diabetic or not using Machine Learning.

The model is trained using the Support Vector Machine (SVM) algorithm on the PIMA Indians Diabetes Dataset.

---

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

---

## ML Workflow
1. Data Collection
2. Data Cleaning
3. Feature Scaling
4. Train-Test Split
5. Model Training
6. Accuracy Evaluation
7. Prediction System

---

## Accuracy
- Training Accuracy: 78.66%
- Testing Accuracy: 77.27%
